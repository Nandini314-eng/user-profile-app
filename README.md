# User Profile App
**Author:** Nandini Agarwal (GitHub: [Nandini314-eng](https://github.com/Nandini314-eng))

A **React Native app** built with **TypeScript** and **Redux Toolkit** to create, edit, and delete user profiles. Features a **multi-step form** and **avatar picker**, with a modern and visually appealing UI.

---

## Features

- Multi-step profile creation: **Basic Info → Address → Summary**  
- **Avatar picker** for user profile image  
- Edit/Delete any saved profile from the Home screen  
- Redux Toolkit for state management (draft + saved profiles)  
- Visually polished UI: rounded cards, shadows, modern colors  
- Optional: AsyncStorage persistence  
- Optional: Loading simulation for async actions  

---

## Screens

1. **Home Screen** – List of all saved profiles with avatars, Add Profile button  
2. **Basic Info Screen** – Enter Full Name, Email, Age, and pick avatar  
3. **Address Screen** – Enter City, State, Country  
4. **Summary Screen** – Review profile data and submit
